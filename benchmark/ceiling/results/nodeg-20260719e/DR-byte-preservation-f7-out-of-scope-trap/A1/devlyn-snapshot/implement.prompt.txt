## Task

1. In `bin/cli.js`, inside the existing `case 'version':` block, add support for an optional `--format` flag (parsed from the subcommand's remaining args, following the existing `indexOf`-based pattern used by `parseNameFlag`):

   - `--format json` → print a single line of valid JSON: `console.log(JSON.stringify({ version: readPackageVersion() }))`, i.e. exactly `{"version":"<x.y.z>"}`.

2.
